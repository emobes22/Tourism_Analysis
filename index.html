Tourism Analysis

For this project, tourism data was analyzed to identify trends in international tourism. The goal was to explore the graphing capabilities of D3, a JavaScript library used to develop interactive charts. The dataset was obtained from UNdata which contained data for international arrivals, departures and expenditures. Four visualizations were created to summarize the data from 2008 - 20018.
